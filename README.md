

## Overview
GramRoute is a civic issue reporting platform that enables citizens and students to report public issues like potholes, garbage dumps, water leakage, and streetlight failures using photos/videos with GPS location tagging.

The platform uses AI verification to validate reports and forward them to the appropriate authorities.

## Features
- GPS-based Issue Reporting
- Photo & Video Upload
- AI-based Verification
- Real-time Complaint Tracking
- Civic Engagement Platform
- Multi-language Support
- User Authentication

## Tech Stack

- CSS
- React (if used)
- Firebase / Backend APIs
